Description
=============
Simple website for school on python 3.10 with django 4.1.3

Stack
=============
- Python 3.10
- Django 4.1.3
- HTML
- CSS
- Docker

How to run
=============
1. Clone the repository from GitHub:   
		`$ git clone https://github.com/Santikasl/school-website.git`
2. Navigate into the project directory:  
		`$ cd school-website`
3. Run container:  
		`$ docker-compose up`
4. Open site:  
		http://localhost:8000/
